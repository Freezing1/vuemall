# 技术栈
前端：Vue，Vue Router，axios，Vuex  
后端：Node.js+Express+MongoDB

# 简介
**系统主要模块：**
1. 商品列表模块
2. 登录注册模块
3. 购物车模块
4. 地址模块
5. 订单确认模块
6. 订单成功模块

# Build Setup
## 安装依赖
npm install

## node 启动 www文件
node vuemall/server/bin/www

## 执行本地本地开发命令 访问localhost:8080
npm run dev

## 打包
npm run build
